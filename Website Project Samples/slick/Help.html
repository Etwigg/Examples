The purpose of this website is to make it easy to find TV shows and movies to watch by searching for specific tags that are curated by the community.


The homepage has a search function where you can add tags to your search to find recommendations.

The tags menu option on the homepage will give a list of common tags you can add to search.

The random menu option will give a random show or movie.

To add tags, leave reviews, and help curate the content of the website you must make an account. If you do not already have an account you can make one through the My Account menu option on the homepage.
